To see a working demo of Ionic's Intune integration for iOS and Android, explore the follow demo links:

- [Angular Intune Demo](https://github.com/ionic-team/demo-intune-angular) repo with support for Capacitor and Cordova.
- [React Intune Demo](https://github.com/ionic-team/demo-intune-react) repo with support for Capacitor and Cordova.

These examples show how to use brokered authentication, acquiring access tokens, and some of the basic API calls available in the integration.
