# Example App Configuration

This example app demonstrates the Intune plugin.

## Setup

Before running the app, you must configure your Microsoft Intune credentials.

### Android
1. Open `android/app/src/main/res/raw/auth_config.json`.
2. Replace `REPLACE_WITH_CLIENT_ID` with your Azure AD Client ID.
3. Replace `REPLACE_WITH_REDIRECT_URI` with your Redirect URI (signature hash).

### iOS
1. Open `ios/App/App/Info.plist`.
2. Replace `REPLACE_WITH_CLIENT_ID` with your Azure AD Client ID.
3. Replace `REPLACE_WITH_REDIRECT_URI` with your Redirect URI.
